Guia do Usuário CKAN

Este guia do usuário aborda o uso da interface web do CKAN para organizar, publicar e encontrar dados. O CKAN também possui uma poderosa API (interface de máquina), que facilita o desenvolvimento de extensões e a integração com outros sistemas de informação.

Alguns recursos da interface web relacionados à administração do site estão disponíveis apenas para usuários com status de administrador do sistema (sysadmin).

O que é CKAN?

CKAN é uma ferramenta para criar sites de dados abertos. Pense nele como um sistema de gerenciamento de conteúdo (CMS) como o WordPress, mas para dados em vez de páginas e posts de blog. Ele ajuda a gerenciar e publicar coleções de dados, sendo utilizado por governos nacionais e locais, instituições de pesquisa e outras organizações que coletam grandes volumes de dados.

Uma vez que seus dados são publicados, os usuários podem usar seus recursos de busca facetada para navegar e encontrar os dados de que precisam, e pré-visualizá-los usando mapas, gráficos e tabelas.

Conjuntos de Dados e Recursos

No CKAN, os dados são publicados em unidades chamadas "conjuntos de dados" (datasets). Um conjunto de dados é uma parcela de dados — por exemplo, as estatísticas de criminalidade de uma região, os números de gastos de um departamento governamental ou leituras de temperatura de várias estações meteorológicas.

Um conjunto de dados contém duas coisas:

  • Metadados: Informações sobre os dados, como título, publicador, data, formatos disponíveis, licença de uso, etc.
  • Recursos: Os próprios dados. Um recurso pode ser uma planilha CSV ou Excel, um arquivo XML, um documento PDF, um arquivo de imagem, etc. O CKAN pode armazenar o recurso internamente ou simplesmente como um link para um local externo na web.

Nota: Em versões antigas do CKAN, os conjuntos de dados eram chamados de "pacotes" (packages). Esse nome ainda pode aparecer em alguns locais, especialmente internamente e em chamadas de API, mas tem o mesmo significado de "conjunto de dados".

Usuários e Organizações

Os usuários do CKAN podem registrar contas e fazer login. Normalmente, o login não é necessário para pesquisar e baixar dados, mas é essencial para todas as funções de publicação.

Cada conjunto de dados geralmente pertence a uma "organização". Uma instância do CKAN pode ter várias organizações. Por exemplo, se o CKAN for usado como um portal de dados por um governo, as organizações podem ser os diferentes ministérios.

Os administradores de uma organização podem adicionar usuários a ela, com diferentes papéis (Membro, Editor, Admin) dependendo do nível de autorização necessário.

Recursos para Publicadores

Adicionar um novo conjunto de dados

Passo 1: Para criar um conjunto de dados, clique em "Conjuntos de Dados" no topo da página e, em seguida, no botão "Adicionar Conjunto de Dados". Alternativamente, acesse a página da sua organização e clique no mesmo botão.

Passo 2: Preencha os metadados básicos:

  • Título: Um título breve e específico.
  • Descrição: Detalhes sobre a origem e o uso dos dados.
  • Etiquetas (Tags): Palavras-chave para ajudar na busca (ex: "população", "crime").
  • Licença: Informação crucial sobre como os dados podem ser usados.
  • Organização: A organização que será proprietária do conjunto de dados.

Passo 3: Clique em "Próximo: Adicionar Dados".

Passo 4: Adicione um ou mais "recursos". Você pode fazer o upload de um arquivo do seu computador ou fornecer um link para os dados na web.

Passo 5: Preencha o Nome, Descrição e Formato do recurso.

Passo 6: Após adicionar todos os recursos, clique em "Próximo: Informações Adicionais".

Passo 7: Defina a visibilidade como "Público" ou "Privado" e adicione informações de autoria.

Passo 8: Clique em "Finalizar". Seu conjunto de dados está criado!

Editar um conjunto de dados

Para editar um conjunto de dados do qual você é membro ou proprietário, vá até a página do conjunto de dados e clique no botão "Editar". Você poderá alterar todos os metadados, visibilidade e outras informações.

Adicionar, excluir e editar recursos

Na página de edição do conjunto de dados, a barra lateral esquerda oferece opções para gerenciar os recursos. Você pode selecionar um recurso existente para editá-lo ou excluí-lo, ou clicar em "Adicionar novo recurso" para incluir mais dados.

Excluir um conjunto de dados

Na página de edição do conjunto de dados, clique no botão "Excluir" e confirme a ação. O conjunto de dados será ocultado, mas não completamente removido. Para exclusão permanente, contate o administrador do site.

Criar uma organização

Se você tiver permissão, pode criar uma nova organização clicando em "Organizações" no menu principal e, em seguida, em "Adicionar Organização". Preencha o nome, a descrição e uma imagem para a página da organização.

Gerenciar uma organização

Se você for um administrador de uma organização, verá um botão "Admin" na página dela. Nesta área, você pode editar as informações da organização e gerenciar seus membros, adicionando usuários e definindo seus papéis (Membro, Editor ou Admin).

Encontrando Dados

Pesquisando no site

Para encontrar conjuntos de dados, digite qualquer combinação de palavras-chave na caixa de pesquisa presente em qualquer página. Os resultados serão exibidos e você poderá usar os filtros na coluna da esquerda para refinar sua busca por etiquetas, formatos de dados, organizações, etc.

Explorando conjuntos de dados

Ao encontrar um conjunto de dados de interesse, a página do conjunto de dados mostrará seus metadados e uma lista de recursos. Muitos tipos de recursos (como CSV, XLS, imagens, PDF) podem ser pré-visualizados diretamente na página do recurso, que também oferece a opção de download.

Guia de Busca: Simples e Avançada

O CKAN suporta dois modos de pesquisa a partir do mesmo campo de busca.

Busca Simples:

Se a sua busca não contiver dois-pontos (:), ela será tratada como uma busca simples. Você pode usar modificadores:

  • censo: Encontra conjuntos de dados com a palavra "censo".
  • censo +2022: Exige que a palavra "2022" esteja presente.
  • censo -2010: Exclui resultados com a palavra "2010".
  • "censo europeu": Busca a frase exata.
Busca Avançada:

Se a sua busca contiver dois-pontos (:), você pode especificar campos para a busca, no formato campo:termo.

  • title:europeu: Busca "europeu" apenas no campo do título.
  • title:europ*: Usa um curinga para encontrar palavras como "europa" e "europeu".
  • title:europa || title:africa: Busca por "europa" OU "africa" no título.
  • author:powell~: Realiza uma busca por proximidade (fuzzy search), encontrando nomes semelhantes como "jowell".

Dica: Para a busca avançada, você pode usar title para o nome do conjunto de dados e text para um campo geral que inclui autor, licença, etiquetas, etc.

Personalização

Você precisa estar logado para usar os recursos de personalização.

Gerenciando seu Feed de Notícias

No topo de qualquer página, clique no ícone do painel (dashboard) ao lado do seu nome para ver seu Feed de Notícias. Ele mostra alterações em conjuntos de dados, organizações e usuários que você "segue". Para seguir um item, basta clicar no botão "Seguir" na página do conjunto de dados ou da organização.

Gerenciando seu Perfil de Usuário

Clique no ícone de engrenagem no topo da página para editar seu perfil. Você pode alterar seu nome completo, e-mail, uma breve biografia e sua senha. As alterações são salvas ao clicar em "Atualizar Perfil".
